# One-Click Cleanup

The One-Click Cleanup feature is designed to keep your development machine lean and fast with minimal effort.

## How it works

1. Devian identifies "safe to delete" files like npm caches, old Docker images, and temporary build files.
2. You can review the list of items to be deleted.
3. Click "Clean All" to reclaim space instantly.

## Automation

In the future, Devian will support scheduled cleanups to ensure you never run out of disk space during a critical build.
