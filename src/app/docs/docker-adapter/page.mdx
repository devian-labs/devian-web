# Docker Adapter

The Docker Adapter integrates seamlessly with your local Docker Desktop installation.

## Capabilities

- **Container Management**: Start, stop, and restart containers.
- **Image Cleanup**: Identify and remove unused or dangling images.
- **Resource Usage**: Monitor CPU and Memory usage for each container.

## Setup

Devian auto-detects your Docker installation. If the Docker command is not found, ensure Docker Desktop is running.
