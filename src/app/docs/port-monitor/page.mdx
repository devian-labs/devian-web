# Port Monitor

The Port Monitor provides real-time visibility into all open network ports on your machine.

## Features

- **Owner identification**: See which process is holding a specific port.
- **Conflict detection**: Get alerted when multiple services try to bind to the same port.
- **One-click kill**: Terminate zombie processes directly from the UI.

## Troubleshooting

If a port is "in use" but you don't know why, use the Port Monitor to track down the hidden process (like a hanging Docker container or a background script).
