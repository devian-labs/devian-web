# Disk Analyzer

Identify what's consuming your local storage and reclaim space in seconds.

## Scan Targets

- **node_modules**: Find heavy dependency folders across multiple projects.
- **Docker Caches**: Identify build caches and unused volumes.
- **Build Artifacts**: Locate `dist`, `target`, and `build` folders.

## Smart Cleanup

Use the Disk Analyzer to bulk-delete unnecessary files safely without affecting your source code.
