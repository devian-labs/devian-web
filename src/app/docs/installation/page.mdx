# Installation

Devian Desktop is currently in Public Beta. You can download the latest version for macOS from our landing page.

## MacOS

1. Download the `.dmg` file.
2. Open the `.dmg` file and drag **Devian** to your Applications folder.
3. Launch the app. You may need to grant permissions for Disk Access and Docker interaction.

## Windows (Coming Soon)

We are actively working on the Windows version. Stay tuned for updates in our [Blog](/blog).

## Minimum Requirements

- MacOS 12.0 or later
- Docker Desktop (recommended for Docker features)
- Node.js / Python / Rust (for repository scanning)
