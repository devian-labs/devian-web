# Introduction

Welcome to the **Devian Desktop** documentation. Devian is the all-in-one development environment manager built for modern developers.

It helps you see, understand, and control everything running on your machine --- from projects and ports to Docker containers and disk usage --- with intelligent automation powered by AI.

## Vision

Give developers full visibility and intelligent control over everything running on their local machine.

**Clarity &rarr; Insight &rarr; Optimization &rarr; Automation**

Devian is not just a monitor. It is a local development intelligence layer.

## Key Features

- **Project Scanner**: Auto-detect local repositories and identify tech stacks.
- **Port Monitor**: View open ports and identify owning processes.
- **Docker Adapter**: Manage running and stopped containers.
- **Disk Analyzer**: Deep scan for large folders and caches.
- **AI Environment Insights**: Plain-language explanations of system state.
